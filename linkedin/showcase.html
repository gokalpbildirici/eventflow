🔄 Event Flow

POST /api/orders → Order Service → Kafka [order.events] → Payment Service → Kafka [payment.events]
Kafka [order.events] → Inventory Service → Kafka [inventory.events]
Kafka Topics → Notification Service → 📋 Logs
Kafka Topics → Analytics Service → 📈 Metrics API

📡 API Showcase

POST/api/orders
Create a new order
// Request { "customerId": "C001", "productId": "P001", "quantity": 2, "totalAmount": 3000.00 } // Response 201 { "id": "uuid", "status": "PENDING", ... }
GET/api/orders
List all orders
// Response 200 [{ "id": "uuid", "customerId": "C001", ... }]
POST/api/products
Add a product
// Request { "id": "P005", "name": "Headset", "stock": 50, "price": 150.00 }
GET/api/analytics/summary
Get business metrics
// Response 200 { "totalOrders": 42, "successfulPayments": 34, "failedPayments": 8, "stockReservations": 40, "stockFailures": 2 }
